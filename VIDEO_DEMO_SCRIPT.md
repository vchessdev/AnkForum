## **PHẦN 1: Poll System (2 phút)**

### Intro
```
"Chào mọi người! Hôm nay mình sẽ hướng dẫn các tính năng mới 
trên AnkForum. Feature đầu tiên là POLL - tính năng bình chọn."
```

### Demo Create Poll
```
Bước 1: Vào trang chủ
  "Bạn thấy ở đây là Post Composer - nơi viết bài."
  "Bấm vào nút 'Poll' ở đây"
  
Bước 2: Form tạo Poll
  "Điền câu hỏi: 'Ngôn ngữ nào bạn thích?'"
  "Thêm lựa chọn: JavaScript, Python, Go"
  "Có thể chọn 'Multiple choice' nếu muốn cho phép chọn nhiều"
  "Chọn hạn: 7 ngày"
  
Bước 3: Tạo Poll
  "Click 'Tạo Poll' → Poll được tạo thành công"
  
Bước 4: Vote on Poll
  "Mọi người có thể bình chọn trực tiếp trên trang chủ"
  "Kết quả cập nhật real-time"
```

### Results
```
"Sau 7 ngày, bạn sẽ có kết quả survey từ cộng đồng"
"Có thể xem phần trăm, số phiếu, và ai đã bình chọn"
```

## **PHẦN 2: Livestream (4-5 phút)**

### Intro
```
"Feature thứ hai: LIVESTREAM - phát trực tiếp từ AnkForum.
Không cần YouTube hay Twitch - tất cả từ trên forum này."
```

### Demo Broadcasting
```
Bước 1: Vào Broadcast Dashboard
  "Click nút 'Livestream' trên trang chủ"
  "Hoặc truy cập /?page=broadcast"
  
Bước 2: Điền thông tin
  "Tiêu đề: 'Tutorial JavaScript Live'"
  "Mô tả: 'Học JavaScript cơ bản'"
  
Bước 3: Bắt đầu stream
  "Click 'Bắt đầu phát trực tiếp'"
  "✅ Stream của bạn LIVE"
  
Bước 4: Dashboard stats
  "Thấy 3 metrics chính:"
  "  • 👥 Viewer count = số người xem (real-time)"
  "  • ⏱️ Duration = thời gian phát (auto-update)"
  "  • 💬 Comments = số bình luận"
  
Bước 5: Share & Promote
  "Copy link → Gửi cho bạn bè"
  "Họ click vào → Xem stream của bạn"
  
Bước 6: Live Chat
  "Khi có viewers chat"
  "Bạn có thể thấy comments real-time"
  "Có thể reply/engage với viewers"
  
Bước 7: End Stream
  "Khi xong, click 'Kết Thúc Stream'"
  "Stream lưu trữ vĩnh viễn"
```

### Demo Watching
```
"Nếu bạn muốn XEM livestream:"

Bước 1: Từ trang chủ
  "Kéo xuống thấy section '🔴 Đang phát trực tiếp'"
  "Click vào stream muốn xem"
  
Bước 2: Watch interface
  "Nhìn thấy:"
  "  • Video player (center)"
  "  • 🔴 LIVE badge + viewer count"
  "  • Stream duration"
  
Bước 3: Chat
  "Bên phải là chat trực tiếp"
  "Nhập tin nhắn → chat với streamer & viewers"
  "Comments công khai cho tất cả"
```

### Live Badge
```
"Khi bạn đang livestream:"
"Navbar sẽ hiển thị '🔴 LIVE' next to your name"
"Nhấp nháy/pulse = đang phát"
"Báo hiệu cho mọi người: BẠN ĐANG ONLINE"
```

  "✅ JPG, PNG, GIF, WebP"

  "Max 500MB / file"
